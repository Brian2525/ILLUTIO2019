¿Qué es la realidad aumentada?

La realidad aumentada es la tecnología que te permite interactuar entre el mundo físico y objetos virtuales creados a partir de una computadora o un smartphone

Para que sirve la realidad aumentada

La realidad aumentada es utilizada cotidianamente en distintas industrias como los juegos, la capacitación, la milicia, el entrenamiento. Se utiliza para mostrar más información o alguna explicación de objetos cotidianos como lo puede ser

Tipos de realidad aumentada

La realidad aumentada se combina generalmente con geolocalización, con reconocimiento de imagen y recientemente con reconocimiento facial

Ejemplo de Realidad aumentada y geolocalización

Un ejemplo ya cotidiano de la realidad aumentada y la geolocalización es Pokemón Go, funciona con la API de google maps y de google places para determinar tanto tu ubicación como las condiciones de los lugares (instalaciones electricas, paradas recurrentes, lugares de tránsito vehicular, escuelas, etc), para determinar donde es mejor posicionar los distintos tipos de Pokemones. Actualmente se puede utilizar también para el turismo, para posicionar anuncios de manera virtual, y se están haciendo pruebas para colocarrealidad aumentada en los autos

Realidad aumentada y reconocimiento de imagen

Un ejemplo de esta tecnología es descargar una app como RA CAM y escanear imagenes para asociarlas con libros, revistas, periodicos que estén asociados a la app para poder mostrar sobrepuestas nuevas imagenes, videos, modelos 3D, animaciones 3D. El uso de esta tecnlogía está asociada al entretenimiento, a la publicidad, a la educación.
